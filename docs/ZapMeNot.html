zap_me_not.model module¶

class zap_me_not.model.Model[source]¶

Bases: object

Performs point-kernel shielding analysis.

The Model class combines various shielding elements to perform the point-kernel photon shielding analysis. These elements include sources, shields, and detectors.

Variables

  • source (zap_me_not.source.Source) – The source distribution (point, line, or volume) included in the model.

  • shield_list (list of zap_me_not.shield.Shield) – A list of shields (including the source volume) contained in the model.

  • detector (zap_me_not.detector.Detector) – The single detector in the model used to determine the exposure.

  • filler_material (zap_me_not.material.Material) – The (optional) material used as fill around the formal shields.

  • buildup_factor_material (zap_me_not.material.Material) – The material used to calculate the exposure buildup factor.
