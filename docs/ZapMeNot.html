Abtract class to model a radiation source.

Variables

points_per_dimension (list of integers) – The number of source points to be used in each dimension when modeling the uniform source distribution throughout the body of the source. Typically a list of three integers for three-dimensional sources, one integer for one diensional sources, and not significant for point sources.
